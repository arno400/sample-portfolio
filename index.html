Projects

1. Interactive Portfolio Website

I designed and developed this very portfolio site using HTML, CSS, and vanilla JavaScript. It features a responsive layout, smooth scrolling, and an interactive project carousel. I focused on performance, achieving a Lighthouse score above 90 and ensuring accessibility with proper ARIA attributes.

Key highlights include a custom carousel component, mobile-first design, and optimized asset loading. The project taught me deeper DOM manipulation techniques and the importance of semantic HTML for SEO and screen readers.

View on GitHub

2. Real-Time Weather Dashboard

Built with React, this app fetches live weather data from the OpenWeatherMap API. I implemented features like location search, unit toggles (°C/°F), and dynamic background changes based on weather conditions.

To enhance performance, I used React’s lazy loading for components and memoization for expensive computations. The UI was designed for clarity, with SVG icons and chart visualizations using Chart.js.

View on GitHub

3. Collaborative Task Manager

This full-stack application allows multiple users to manage tasks in real time. Front-end is React with Context API; back-end is Node.js and Socket.IO for live updates.

Features include authentication, role-based permissions, and drag-and-drop task boards. I integrated MongoDB for data persistence and deployed the app on Heroku with CI/CD via GitHub Actions.

View on GitHub

4. E-Commerce Platform Prototype

A prototype storefront built with Next.js and Stripe API integration. Users can browse products, add items to cart, and complete mock checkouts. I implemented server-side rendering for faster initial loads and SEO optimization.

The admin dashboard includes product management and order tracking. I also set up automated email notifications using Nodemailer.

View on GitHub

5. Blogging Platform with Comments

A MERN-stack application featuring user authentication, rich-text post editor, and threaded comments. I used Express.js for the API, MongoDB Atlas for the database, and deployed via Netlify and Heroku.

I prioritized security with JWT tokens, input sanitization, and rate limiting. The UI includes live markdown previews and image uploads to Cloudinary.
